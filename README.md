## ⚡ Features

✅ Voice recognition via microphone  
✅ Text-to-speech responses  
✅ Tells current time on command  
✅ Custom greetings  
✅ Continuous command loop  
✅ Quickstart demo script (no mic needed)  
✅ Clean, modular code structure
